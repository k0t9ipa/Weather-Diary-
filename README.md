# Weather-Diary-

Автор: Гладков Константин Александрович

Позволяет добавлять, просматривать и фильтровать записи о погоде.

Weather Diary — это настольное приложение с графическим интерфейсом, предназначенное для ведения личного дневника погоды. Программа позволяет фиксировать, просматривать, фильтровать и сохранять данные о погоде, а также обеспечивает удобный обмен проектом с помощью системы контроля версий Git.

Основные возможности
Добавление записейПользователь может ввести дату, температуру, краткое описание погоды и отметить, были ли осадки. Все поля проходят автоматическую валидацию: дата должна быть в формате ДД.ММ.ГГГГ, температура — числом, описание не может быть пустым.
Просмотр и фильтрацияЗаписи отображаются в виде таблицы. Доступна фильтрация по дате (например, показать все записи за выбранный день) и по температуре (например, вывести только записи с температурой выше +10°C).
Сохранение и загрузка данныхВсе записи хранятся в формате JSON. Пользователь может сохранить текущие данные в файл и загрузить их обратно при запуске программы.
Использование GitПроект сопровождается локальным Git-репозиторием. В корне проекта размещён файл .gitignore для исключения временных и системных файлов. Проект опубликован на GitHub для совместной работы и хранения истории изменений.
ДокументацияВ репозитории размещён файл README (в формате .md), где указаны:
имя и фамилия автора,
краткое описание программы,
инструкция по запуску,
примеры использования.
Технические детали
ИнтерфейсРеализован на Python с использованием библиотеки Tkinter. Для удобства ввода даты можно использовать модуль tkcalendar.
Хранение данныхИспользуется стандартный формат JSON. Каждая запись — это словарь с полями: дата, температура, описание, осадки (да/нет).
ВалидацияПроверка корректности ввода реализована на уровне интерфейса: при ошибке пользователь получает сообщение и не может добавить некорректную запись.
ФильтрацияРеализованы отдельные функции для фильтрации по дате и температуре. Результаты отображаются в таблице.
Git и GitHubПроект инициализирован как Git-репозиторий, добавлен .gitignore для Python, проект выложен на GitHub с подробным README.
Пример использования
Запустить приложение.
Ввести дату, температуру, описание погоды и отметить осадки.
Нажать «Добавить запись» — запись появится в таблице.
Использовать фильтры для поиска нужных записей.
Сохранить данные в JSON-файл или загрузить ранее сохранённые записи.
Все изменения фиксируются в Git и синхронизируются с GitHub.
